Call C++ native library on Linux

1. write c++ code

2. make shared library (*.so) on Linux
	gcc -shared -fPIC -m64 -o foge.so foge.cpp
	'cp' -f foge.so publish/

3. copy self-contained package to linux
	unzip ~/publish.zip  -d .
	cp foge.so publish/

4. chmod execute binary
	chmod +x publish/CoreConsole

5. run
	./publish/CoreConsole
